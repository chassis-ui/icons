## Icon Usage

Icons can be used in multiple formats:

### SVG Icons

Individual SVG files for maximum flexibility:

```html
<svg class="icon" aria-hidden="true">
  <use href="/chassis-icons.svg#icon-home"></use>
</svg>
```

### Icon Font

Use the icon font for text-based implementation:

```css
.icon-home::before {
  font-family: 'chassis-icons';
  content: '\e001';
}
```
